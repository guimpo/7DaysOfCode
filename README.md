# 7DaysOfCode

## How to run:

#### Dependencies:

- pyenv
- anaconda3-2023.03

#### Activate anaconda3-2023.03:

```shell
$ pyenv local
```

#### Run jupyter-lab:

```shell
$ jupyter-lab
```
#### Download file:

- [Spotify dataset](https://www.kaggle.com/datasets/maharshipandya/-spotify-tracks-dataset/download?datasetVersionNumber=1)
